ABOUT THE PROJECT

Key features of our project

Reduced Manpower

Current system is totaly dependent on human. So we have designed the system in a way to reduce human intervention.

Time Saving

It is a waste of human efforts for this type of simple work.The proposed system saves the customers time by completing the work with just one click.

Reduced Water Wastage

Current system wastes a lot of water but here, the water is filtered and reused for further purposes.

Affordable

The proposed system is much cheaper than already available systems and affordable by anyone.

Description:

I-panels is a smart system which can be operated by a smart device remotely, and the main purpose of this system is to clean solar panels smartly. It will clean the solar pannels by it self by just giving a simple command from your smart device.
